Alcuni script PHP di esempio

Per (1-6) v. slide set 1; per (7-8) v. slide set 2

Istruzioni per l'esecuzione

  1. oneline.php [src] – file php di un rigo (vedi slide)
  2. primo.php [src] – primo esempio nelle slide

    3. primo.html [src] – stesso contenuto di primo.php ma, a causa dell'estensione .html, non va all'engine PHP e l'output è vuoto (vedi slide)
  3. embedded.php [src] – esempio di piccolo script PHP inserito (embedded) in html
  4. pure.php [src] – genera lo stesso rendering di embedded.php ma solo con PHP “puro” senza HTML

    5. pure_nl.php [src] – variante con new lines; genera lo stesso HTML di embedded.php
    pure3.php [src] – variante con 3 echo()  pure1.php [src] – variante con un solo echo()
  5. echo_multiline.php [src] – alcune particolarità dell'argomento di echo
  6. phpinfo.php [src] – info da interprete/server PHP su se stesso
  7. escape.php [src] – escape da HTML a PHP a HTML e di nuovo PHP (preservato stato $x)

    8. escape1.php [src] – escape da HTML a PHP a HTML e di nuovo PHP (preservato stato $x)
    escape_if_1.php [src] – interazione tra if php e HTML inframmezzato
    escape_if_2.php [src] – interazione tra if php e HTML inframmezzato
    escape_for.php [src] – interazione tra for php e HTML inframmezzato
    escape_for_1.php [src] – interazione tra for php e HTML inframmezzato
    escape_mix.php [src] – interazione tra PHP (vari costrutti) e HTML inframmezzato
  8. eqtag0.php [src] – uso del tag abbreviato <?=

    9. eqtag1.php [src] – uso del tag abbreviato <?=
    eqtag2.php [src] – uso del tag abbreviato <?=
    eqtag_no_out.php [src] – uso “apparente” del tag abbreviato <?=
  9. hello.html [src] – scrive Hello world! NB: aprirlo nel browser equivale a inviare GET /hello.html

    10. hello_get_post.html [src] – invia GET o POST (mostrando che hanno lo stesso effetto) al path ./hello.html
  10. form0_post.php [src] – questo script visualizza un form che invia un POST verso se stesso; NB: tutto lo script è attivato sia per GET che per POST

    11. form0_post_error_rep.php [src] – come form0_post.php ma sopprime warning per parametro non definito nella richiesta POST
    form0_get.php [src] – come form0_post.php ma il form usa GET senza protezione per invio dato
    form0_get_protect.php [src] – come form0_get.php ma il form usa GET con protezione per invio dato
    form1_post.php [src] – script con gestione specifica per le richieste POST
    form2_post.php [src] – script più complesso, con gestione specifica per le richieste POST
    form_get_post.php [src] – script con gestioni specifiche per le richieste GET e per le richieste POST
  11. gestione_form.php [src] – gestione più elaborata dei form

    12. validate.php [src] – validazione dell'input nei form; N.B.: dopo submit/POST i form vengono cancellati (v. https://www.w3schools.com/php/php_form_validation.asp e seguenti)


    validate_keep.php [src] – validazione dell'input nei form; N.B.: dopo submit/POST nei form rimane l'input dell'utente (v. https://www.w3schools.com/php/php_form_complete.asp)
  12. attacks.html [src] – su attacchi XSS/CSRF e CORS (solo doc; non codice per ora)
  13. upload_test.html [src] – form cliente per attivare upload.php

    14. upload.php [src] – (link a sinistra non cliccabile) script per validazione e upload di file dal cliente verso il server
  14. download_bin.php [src] – script per download del file binario small.mp4 dal server sul client
  15. include_ex.php [src] – illustra uso di include(), includendo footer.php

    16. include_once_ex.php [src] – illustra uso di include_once(), includendo footer.php
    include_require_ex.php [src] – fallimento di include() e require()
    include_require_ex_1.php [src] – fallimento di include(), require() e livello di error reporting
  16. create1.php [src] – crea un nuovo file newfile.txt (esegui prima rm newfile.txt)

    17. create2.php [src] – come create1.php, osservare effetto se newfile.txt esiste ed errore se newfile.txt è protetto da scrittura
  17. html_namespace_test.php [src] – usa i namespace definiti in html_namespace.php

    18. html_namespace.php [src] – (link a sinistra non cliccabile) illustra uso dei namespace
  18. imagick_draw.php [src] – demo per il modulo PHP Imagick (funziona solo dopo aver installato il modulo con pecl install imagick)
